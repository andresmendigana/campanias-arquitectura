# Anexo A - Glosario

| Término | Definición |
|---------|-----------|
| ACH | Automated Clearing House. En Colombia, operador de PSE |
| ADR | Architecture Decision Record. Registro de decisión arquitectónica |
| API Gateway | Punto de entrada único para APIs, maneja routing, auth y rate limiting |
| Apetito de riesgo | Nivel de riesgo que el banco está dispuesto a aceptar |
| AS-IS | Estado actual del sistema |
| BPM | Business Process Management. Gestión de procesos de negocio |
| C4 Model | Modelo de documentación de arquitectura (Context, Container, Component, Code) |
| Circuit Breaker | Patrón que previene llamadas a servicios fallidos |
| CQRS | Command Query Responsibility Segregation |
| CRM | Customer Relationship Management |
| Datacrédito | Central de información crediticia en Colombia |
| Descompensación contable | Diferencia entre lo registrado y lo real en cuentas |
| DLQ | Dead Letter Queue. Cola para mensajes que no pudieron procesarse |
| DynamoDB | Base de datos NoSQL serverless de AWS |
| ESB | Enterprise Service Bus. Bus de servicios empresarial |
| Event Sourcing | Patrón donde el estado se reconstruye desde eventos |
| EventBridge | Servicio de bus de eventos serverless de AWS |
| Habeas Data | Ley colombiana de protección de datos personales (Ley 1581/2012) |
| Host Transaccional | Core bancario donde se procesan transacciones |
| IAM | Identity and Access Management |
| Kinesis Firehose | Servicio de streaming de datos de AWS |
| KMS | Key Management Service. Gestión de llaves de cifrado |
| Lambda | Servicio de compute serverless de AWS |
| Libre destino | Tipo de crédito sin destinación específica |
| Looker | Herramienta de BI y dashboards de Google Cloud |
| Microcrédito | Crédito de bajo monto para personas sin acceso a banca tradicional |
| mTLS | Mutual TLS. Autenticación mutua con certificados |
| MVP | Minimum Viable Product. Producto mínimo viable |
| PII | Personally Identifiable Information. Datos personales |
| PSE | Pagos Seguros en Línea. Plataforma de pagos interbancarios en Colombia |
| Push Notification | Mensaje enviado directamente al dispositivo móvil del cliente |
| RBAC | Role-Based Access Control |
| Re-platform | Migrar una aplicación a una nueva plataforma tecnológica |
| Rotativo | Crédito con cupo revolvente (se puede reutilizar lo pagado) |
| Scoring | Calificación de riesgo crediticio de un cliente |
| Serverless | Modelo donde el proveedor cloud gestiona la infraestructura |
| SFC | Superintendencia Financiera de Colombia |
| SQS | Simple Queue Service. Servicio de colas de AWS |
| Step Functions | Servicio de orquestación de workflows de AWS |
| TO-BE | Estado futuro/propuesto del sistema |
| VPC | Virtual Private Cloud. Red privada virtual |
| WAF | Web Application Firewall |
